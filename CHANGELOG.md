# Changelog

All notable changes to AegisMemory will be documented in this file.

## [1.0.0] - 2026-02-16

### Added
- Initial release of AegisMemory
- Encrypted memory storage with AES-256-GCM
- X1 Vault (IPFS) integration for off-chain storage
- X1 Blockchain anchoring via Memo Program
- TOON format for 46% space savings
- Durable JSONL queue with retry logic
- Persistent state management
- OpenClaw plugin integration
- CLI tool for status and management
- Configurable agent IDs
- Multiple RPC fallback support
- Memory recall functionality
- Automatic queue processing

### Security
- Wallet-based key derivation
- End-to-end encryption
- On-chain integrity verification
- Privacy-preserving design

### Performance
- 0.002 XNT per memory anchor
- ~500ms total latency
- 25% compute unit usage
- Free IPFS storage

### Documentation
- Comprehensive README
- Storage architecture guide
- Live test results
- API documentation
- Contributing guidelines
